# Planejador-financeiro

Um sistema web para registrar receitas e despesa, acompanhar o saldo e organizar as finanças de forma simples.

## Funcionalidades:

- adicionar receitas ou despesas.
- categorizar lamçamentos.
- editar registros.
- excluir registros.
- filtrar por tipo.
- filtrar por categoria.
- exibir resumo financeiro.
- salvar dados (local Storage).


## tecnologias ultilizadas:

- HTML5
- CSS3
- JavaScript (ES6)

### Principais conceitos de JS usados:

```
- Manipulação do DOM
- Delegação de eventos
- Estruturas de dados do JavaScript
- Local Storage
- criação dinâmicas de tabelas via JS
- responsividade para mobile
```
